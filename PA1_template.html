Imputing missing values

  1. Total number of missing values in the dataset (the total number of rows with NAs) = 2304
length(which(is.na(activity$steps)))
## [1] 2304
  1. Filling in all of the missing values in the dataset by the mean for that 5-minute interval
uni.interval<-unique(activity$interval[which(is.na(activity$steps))])
activity1<-activity
for (i in uni.interval) {
activity1$steps[which(is.na(activity$steps)&activity$interval==i)]<-avg.steps[as.numeric(row.names(avg.steps))==i]
}

  1. activity1 above is a new dataset that is equal to the original dataset but with the missing data filled in

  2. Histogram of the total number of steps taken per day and the mean and median of total number of steps taken per day. In contrast to the original dataset, imputing missing data has shifted up the mean and median of total number of steps taken per day.
